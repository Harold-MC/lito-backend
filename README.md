## Description

Para el proyecto utilize como motor de base de datos Sqlite, pero la configuracion realizada con el ORM, permite actualizar el driver de la
conexion para adaptarse facilmente a otro motor de base de datos.

Por defecto la base de datos utilizada ya tiene algunos registros; en el modulo de base de datos esta la configuracion del seeder y la migracion
ejecutadas

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
